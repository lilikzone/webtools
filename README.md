# ibase-webtools-react
IBASE Frontend with REACT

#Prepare Your Server
1. Install node js
  Ubuntu
  $ apt-get install nodejs
  Centos
  $ yum install nodejs
2. Install npm  
  $ apt-get install npm
  $ yum install npm
3. Upgrade nodeJs to latest Version
  $ sudo npm cache clean -f
  $ sudo npm install -g n
  $ sudo n 10.15.0   (exp. we use node js 10.* version
 4. Check Version
  $ node -v
  $ npm -v
